# Scroll Up Without the Arrow Button (Manual Scroll)

**URL:** https://www.automationexercise.com  
**Flow:** Scroll down to the bottom of the page and scroll back to the top manually (without using the floating arrow button).

## Steps

1. Navigate to `https://www.automationexercise.com`.
2. Verify the page has loaded and the hero/carousel section is visible at the top.
3. Scroll down to the bottom of the page (to the footer/subscription section).
4. Verify the footer content and subscription section are visible.
5. Using keyboard or mouse wheel, scroll back up to the top of the page manually (do NOT click the scroll-up arrow button).
6. Verify the page returns to the top.
7. Verify the **"Full-Fledged practice website for Automation Engineers"** text or the carousel is visible at the top.

## Expected Result

- The page can be scrolled back to the top manually.
- The hero/carousel section is visible after manual scroll-up.
- The page functions normally without relying on the floating scroll button.
