# Verify Test Cases Page

**URL:** https://www.automationexercise.com/test_cases  
**Flow:** Navigate to the Test Cases page and verify it loads correctly with all test cases listed.

## Steps

1. Navigate to `https://www.automationexercise.com`.
2. Click the **"Test Cases"** link in the top navigation bar.
3. Verify the URL changes to `https://www.automationexercise.com/test_cases`.
4. Verify the **"Test Cases"** page heading is visible.
5. Verify the page lists all 26 test cases, each with:
   - A test case number/ID
   - A descriptive title
6. Spot-check a few entries, for example:
   - Test Case 1: Register User
   - Test Case 9: Search Product
   - Test Case 26: Verify Scroll Up without 'Arrow' button and Scroll Down functionality

## Expected Result

- Test Cases page loads without errors.
- All 26 test cases are listed with their IDs and titles.
- Page navigation functions correctly from the main nav.
