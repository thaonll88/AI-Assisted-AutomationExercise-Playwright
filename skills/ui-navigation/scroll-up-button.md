# Scroll Up Using the Arrow Button

**URL:** https://www.automationexercise.com  
**Flow:** Scroll down to the bottom of the page and use the floating scroll-up arrow button to return to the top.

## Steps

1. Navigate to `https://www.automationexercise.com`.
2. Verify the page has loaded and the hero/carousel section is visible at the top.
3. Scroll down to the bottom of the page (to the footer/subscription section).
4. Verify a **scroll-up arrow button** (↑) becomes visible — typically a floating button in the bottom-right corner.
5. Click the **scroll-up arrow button**.
6. Verify the page scrolls back to the top.
7. Verify the **"Full-Fledged practice website for Automation Engineers"** text or the carousel is visible at the top of the page.

## Expected Result

- The scroll-up button appears after scrolling down.
- Clicking it returns the user to the top of the page smoothly.
- The top-of-page content (carousel/hero section) is visible after scrolling up.
